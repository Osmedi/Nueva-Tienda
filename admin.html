Configuración inicial de Firebase
Pasos para crear tu proyecto Firebase:
1. Ve a console.firebase.google.com
2. Clic en "Agregar proyecto" → ponle un nombre → crear
3. En el menú lateral ve a Firestore Database → Crear base de datos → Modo producción
4. Ve a Configuración del proyecto (ícono ⚙️) → "Tus aplicaciones" → Web (</>)
5. Registra la app y copia las credenciales abajo 👇
Tus credenciales se guardan solo en este navegador (localStorage).
Nadie más puede verlas.
